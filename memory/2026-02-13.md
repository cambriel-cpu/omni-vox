## Dashboard Login Finally Working (Evening)
- Root cause of login not working: CSS `display: flex` on `#login-screen` overrides HTML `hidden` attribute
- Fix: `#login-screen[hidden] { display: none !important; }`
- Debug panel approach (inline JS logging to visible div) was essential for mobile debugging without dev tools
- `sed` for removing debug code was catastrophic — deleted catch blocks, render calls, fetchAll invocations
- **Lesson: NEVER use sed to remove debug code — always do it manually or with precise edits**
