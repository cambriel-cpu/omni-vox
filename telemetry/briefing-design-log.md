# Briefing Email Design Log

Track design iterations and feedback to evolve the morning briefing email.

## Design Principles
- NOT generic — should feel distinctly Omni/Omnissiah
- Dark, mechanical aesthetic inspired by the machine spirit theme
- Clean typography, strong visual hierarchy
- Mobile-friendly
- Evolve something new each day — don't repeat the same template
